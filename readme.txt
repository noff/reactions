Базовая идея: любой интеллект состоит из базовых реакций. 

Базовая реакция: Воздействие => Ощущение => Реакция => Обучение.
Пример: Ожог => Больно => Одернуть руку => Запомнить, что горячее больно

Это крупная реакция. Она также может быть разложена на более малые реакции. Например, одна из составляющих:
Нагрев кожи => Понимаем, что потом коже будет плохо => Лучше предотвратить ожог, поэтому сообщаем, что больно => Запоминаем, что сильный нагрев кожи - больно.

После каждого обучения накапливается статистическая информация, влияющая в будущем на то, какая реакция будет выбираться из тех, которые уже есть либо будет формироваться новая реакция.
